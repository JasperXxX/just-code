Install
  1. Homebrewprereq
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Ghosttyterminal
    brew install --cask ghostty
  3. Node + CLI toolsruntime
    brew install node gh uv tmux
  4. Claude Codecli agent
    npm i -g @anthropic-ai/claude-code
  5. Zededitor
    brew install --cask zed
  6. Spokenlyvoice input
    Mac App Store → "Spokenly"
  7. Raycast + Rectanglelauncher · windows
    brew install --cask raycast rectangle
  8. Claude Desktopoptional gui
    brew install --cask claude
  9. Clone this repoconfigs
    git clone <repo-url> ~/claude-code-loadout
